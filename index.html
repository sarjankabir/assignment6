Using the magnifier

I have tested the Windows Magnifier app on the CNN News website. Pressing the Windows logo keyâ€Œ + Plus sign (+) on the keyboard turns the Magnifier on. Pressing Windows logo key + Esc turns it off. When Magnifier is on, zoom in and out can be done by pressing the Windows logo key + Plus sign (+) or Windows logo key + Minus sign (-). You can also zoom in and out using the mouse by pressing Ctrl + Alt and rotating the wheel on your mouse. The Magnifier is easy to use and has different magnification settings, which is very useful. I have used 200% of magnification. There is also a high contrast mode in the settings, that makes the background dark and the text light. This is another useful feature . I also found that Magnifier provides three options for tracking elements on the screen: text insertion point, mouse pointer, and keyboard focus, the last one being more useful as since keyboard focus can be very helpful when accessing pull-down menus and other controls via the keyboard. Magnifier is easier to use via its onscreen user interface, which makes it especially user-friendly for mouse users. However, I did find it challenging to move through the screen even to finish reading one sentence; it becomes tiresome. Also, magnification sometimes makes the alphabets blurred. I also found myself frequently going out of focus while navigating through the magnifier.

Low vision modes

I have included two kinds of low vision modes in this project, magnification mode and contrast alteration mode. At the top of the web page, two distinct buttons have been added to enable each of these modes. The button labelled 'Magnify' makes the fonts on the page larger The other button labelled 'Alter contrast' changes the background and text color.Each button also has distinct icons to make them distinct for those who have difficulties in differentiating colors.
